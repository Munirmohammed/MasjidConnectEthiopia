# MasjidConnect Ethiopia 🌟

A premium, full-featured Islamic community mobile application designed for Ethiopia. Built with React Native and Expo.

## Features

- **Mosque Map:** Interactive map of ~50 prominent Ethiopian mosques with real GPS coordinates and details.
- **Prayer Times:** High-precision Athan calculations using Adhan.js based on user location.
- **Imam Directory:** Searchable scholarly directory with contact details for prominent Ethiopian Imams.
- **Events:** Upcoming Islamic gatherings, lectures, and Ramadan/Eid programs in Ethiopia.
- **Friday Khutbahs:** Built-in audio player to stream Friday sermon recordings.
- **Community noticeboard:** Lost & Found, Job listings, and general announcements.
- **Donations Portal:** Fundraising campaigns for mosque renovations and community aid with progress tracking.

## Tech Stack

- **Framework:** Expo (React Native)
- **Navigation:** Expo Router (File-based)
- **State Management:** Zustand
- **Calculation:** Adhan.js, date-fns
- **UI/UX:** Custom Emerald & Gold palette with dynamic Dark/Light mode support.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

2. **Run the app:**
   ```bash
   npx expo start
   ```

3. **Open on your device:**
   Scan the QR code in the terminal using the **Expo Go** app (available on iOS and Android).
